## Skills & Expertise

**Data Analysis & Statistics:** Exploratory data analysis, hypothesis testing 
(ANOVA, Kruskal-Wallis, Mann-Whitney U), confidence intervals, regression modeling, 
and statistical inference. Grounded in the frameworks from *The Art of Statistics* 
(Spiegelhalter) and *Data Science for Business* (Provost & Fawcett).

**Machine Learning:** Supervised learning with tree-based models (LightGBM, 
Random Forest, Decision Tree), model evaluation and diagnostics, feature importance 
interpretation, and hyperparameter tuning. Experience with time series forecasting 
using walk-forward validation to prevent data leakage.

**Feature Engineering:** Lag features, rolling statistics, target encoding, 
calendar features, and log transformations for time series and tabular data.

**MLOps & Cloud:** Azure ML model registry, managed online endpoints, scoring 
scripts, and blue/green deployment. Certified Azure Data Scientist Associate (DP-100).

**Python Automation:** Automated reporting pipelines that fetch data from APIs, 
compute KPIs, generate visualizations, and produce HTML dashboards — all from a 
single command.

**Market Basket Analysis:** Co-occurrence analysis using lift, support, confidence, 
and leverage following the Provost & Fawcett framework.

**Data Visualization:** Chart design following *Storytelling with Data* principles — 
small multiples, lollipop charts, heatmaps, and direct labeling. Proficient in 
Matplotlib, Seaborn, Tableau, and Looker Studio.
